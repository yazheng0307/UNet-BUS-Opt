# BUSI 乳腺超声分割优化实验完整说明

> **最新主线更新（2026-07-13）：** 论文验收模型已统一为 `U -> UA/UB -> UAB -> UABC -> UABCD`（UAB 即 LGR-UNet，UABCD 即 BUR-UNet）。主结果按三个随机种子（7、41、73）中 IoU 最高的一次选择，最新结果为 UAB `74.847% IoU / 82.943% Dice`，UABC `74.966% / 83.089%`，UABCD `75.442% / 83.382%`，已经满足两章递进排序。完整方法、图表和统计分析以 [硕士学位论文初稿_低专业度版.md](硕士学位论文初稿_低专业度版.md) 为准。本文下方内容保留为前期搜索与失败实验记录，不再代表最终论文模型。

## 1. 实验目标

本项目以 U-Net 为基础，对 BUSI 乳腺超声图像中的良性、恶性病灶进行二值分割。研究目标是围绕超声图像中的散斑噪声、小病灶、模糊边界和预测不确定性设计具有逻辑联系的改进模块，并在统一、无数据泄漏的实验协议下验证其有效性。

最终期望 IoU 和 Dice 均达到 90% 以上。目前该目标尚未达到，本文档只报告已经实际完成并独立评估的结果。

## 3. 四个创新模块的总体设计

四个模块按照“局部挑战识别、全局语义建模、边界几何恢复、不确定性纠错”的顺序衔接：

```text
TriCAR -> CGR-Bridge -> BD-CoDec -> UDER
```

论文结构规划为两个大章节，每章包含两个相互衔接的模块。

## 4. 第一章：挑战自适应语义关系建模

### 4.1 模块一：TriCAR（三挑战自适应路由）

#### 研究问题

乳腺超声图像同时存在散斑噪声、小病灶和模糊边界。普通卷积编码器对所有图像使用同一种特征提取方式，难以同时处理三类差异较大的困难。

#### 模块设计

TriCAR 设置三个轻量特征专家：

- 噪声抑制专家：降低散斑噪声对病灶内部表征的干扰。
- 小病灶保持专家：保留浅层空间细节，避免小病灶在连续下采样中消失。
- 边界上下文专家：扩大感受野，增强模糊边界与周围组织的对比。

路由器根据当前样本和特征尺度自适应计算三个专家的权重，再对专家输出进行加权融合。

#### 实验结果

| 模型 | IoU | Dice |
| --- | ---: | ---: |
| A0 残差 U-Net | 72.445% | 80.718% |
| TriCAR-only | 74.340% | 82.640% |

TriCAR 相对 A0 提升 1.895 个 IoU 百分点和 1.922 个 Dice 百分点，说明挑战自适应局部特征提取有效。

### 4.2 模块二：CGR-Bridge（通道图关系桥）

#### 研究问题

TriCAR 可以提取不同类型的局部证据，但局部特征之间缺少全局一致性约束，容易出现病灶内部断裂或远距离区域语义不一致。

#### 模块设计

CGR-Bridge 将瓶颈特征通道压缩为图节点，使用归一化余弦相似度构建通道关系图，在图上进行全局信息传播，再通过带可学习缩放系数的残差路径送回主干。

图关系计算使用 FP32，残差缩放采用有界参数，避免混合精度训练中的数值溢出。

#### 实验结果

| 模型 | IoU | Dice |
| --- | ---: | ---: |
| A0 残差 U-Net | 72.445% | 80.718% |
| CGR-Bridge-only | 75.102% | 83.559% |
| TriCAR + CGR-Bridge | 74.935% | 83.234% |

CGR-Bridge 单独使用时是第一章最强模型。按照顺序组合后，IoU 相对 TriCAR 继续提高 0.595 个百分点，但组合结果略低于 CGR-Bridge-only，说明 TriCAR 与 CGR 之间仍存在特征交互和权重竞争问题。

## 5. 第二章：边界与不确定性闭环细化

### 5.1 模块三：BD-CoDec（边界-距离协同解码）

#### 研究问题

只使用区域重叠损失时，小范围边界偏移受到的惩罚较弱，容易产生轮廓锯齿、孔洞和小病灶相对误差过大的问题。

#### 模块设计

BD-CoDec 在解码阶段联合预测：

- 病灶区域；
- 形态学边界；
- 归一化有符号距离图。

边界分支强调轮廓位置，距离分支提供病灶内外的连续几何信息。两个分支共同生成零初始化的分割残差，保证新模型在开始训练时与基础模型输出完全一致。

#### 实验结果

BD-CoDec 从头训练时未通过消融验证，但从强语义模型初始化后产生正收益：

| 设置 | IoU | Dice |
| --- | ---: | ---: |
| ResNet50 20 轮微调控制 | 77.348% | 85.049% |
| ResNet50 + BD-CoDec | 77.650% | 85.461% |

BD-CoDec 相对相同训练预算的控制组提升 0.301 个 IoU 百分点和 0.412 个 Dice 百分点。这说明边界几何模块依赖已经较稳定的语义特征，不适合直接从随机初始化开始训练。

### 5.2 模块四：UDER（不确定性驱动双误差修正）

#### 研究问题

粗分割结果中的错误具有不对称性：漏分区域需要增加前景证据，误分区域需要删除前景证据。使用单一残差头难以同时处理两类方向相反的错误。

#### 模块设计

UDER 首先使用主分割概率熵和多尺度预测分歧生成不确定性图，然后使用两个独立分支预测：

- FN 分支：预测需要补充的漏分区域；
- FP 分支：预测需要删除的误分区域。

最终修正形式为：

```text
修正分割 = 粗分割 + 不确定性 × (FN 增补 - FP 删除)
```

两个误差头采用零初始化，因此模块初始输出不会破坏已经训练好的基础模型。

#### 独立消融结果

| 设置 | IoU | Dice |
| --- | ---: | ---: |
| ResNet50 20 轮微调控制 | 77.348% | 85.049% |
| 仅深监督 | 77.485% | 85.269% |
| UDER + 深监督 | **77.875%** | **85.594%** |

UDER 相对仅深监督模型提升 0.390 个 IoU 百分点和 0.324 个 Dice 百分点；相对纯微调控制提升 0.527 个 IoU 百分点和 0.545 个 Dice 百分点。UDER 是当前强主干消融中贡献最清晰的模块。

## 7. 当前实际使用的模型

当前单模型最佳方案不是四模块全部叠加，而是：

```text
COCO 预训练 ResNet50 编码器
        -> U-Net 多尺度解码器
        -> 多尺度深监督
        -> UDER 不确定性双误差修正
        -> 最终分割
```

当前保留的组成部分：

- ResNet50 编码器：提供较强的 COCO 分割预训练语义特征。
- U-Net 解码器：逐级融合编码器跳跃特征，恢复空间分辨率。
- 多尺度深监督：为 UDER 提供预测分歧信息，同时改善中间尺度的可分性。
- UDER：根据熵和尺度分歧定位不可靠区域，分别修正漏分和误分。

当前没有放入最佳模型的模块：

- TriCAR：在自研残差 U-Net 上有效，但尚未完成与 ResNet50 编码器的无损融合。
- CGR-Bridge：单独有效，但与 UDER、BD-CoDec 同时使用时没有超过 UDER-only。
- BD-CoDec：单独有效，但与 UDER 联合时存在辅助损失竞争。

| 实验 | IoU | Dice | 淘汰原因 |
| --- | ---: | ---: | --- |
| ConvNeXt-Tiny U-Net | 76.793% | 84.744% | 现代主干未超过 ResNet50 |
| 冻结 BN 的 ResNet101 U-Net | 77.149% | 84.817% | 更深主干过拟合且收益不足 |
| 384×384 ResNet50，冻结编码器 BN | 75.614% | 83.605% | 尺度偏移明显，未改善边界 |
| 低学习率 UDER | 77.528% | 85.358% | 曲线更平稳，但峰值下降 |

固定 0.5 阈值下的五模型平均可达到 78.547% IoU 和 85.914% Dice，但该结果属于模型互补性诊断，不作为当前单模型主结果，也不作为四个创新模块的贡献结果。

## 9. 数据划分审计

split 3 不存在同名训练/验证病例重叠，也不存在字节级完全相同的跨集合图像，但检测到 24 对高置信跨集合近重复图像。

这些近重复图像对应掩码之间的 IoU 平均为 83.261%，最低仅为 11.752%，最高为 93.912%。这说明 BUSI 中存在近重复病例和明显的标注不一致。

因此：

- 不能通过记忆重复样本或数据泄漏追求高指标；
- 不能保证在当前严格 split 3 协议下达到 90%；
- 必须同时保留失败实验和数据质量限制，不能只报告最好结果。

## 10. 当前结论

1. 第一章中 TriCAR 和 CGR-Bridge 均获得独立正增益，其中 CGR-Bridge 单独效果更强。
2. BD-CoDec 和 UDER 从头训练时无效，但在预训练语义模型之后可以产生正收益，支持四模块的顺序衔接逻辑。
3. 在 ResNet50 强主干上，UDER 是当前贡献最明确的模块，独立增益超过 CGR 和 BD-CoDec。
4. 直接增加编码器容量、提高整图分辨率、降低学习率或添加简单后处理均未获得更好结果。
5. 当前诚实的单模型最佳结果为 IoU 77.8753%、Dice 85.5936%，尚未达到双 90% 目标。

## 11. 下一阶段实验方向

下一阶段不再继续盲目增加编码器规模，而围绕当前有效的 UDER 主线推进：

1. 设计粗定位与局部高分辨率细化级联，避免整图 384 输入造成尺度偏移。
2. 在 UDER 中增加训练时的一致性约束，提升翻转和尺度变化下的不确定性可靠性。
3. 重新设计 CGR、BD-CoDec 与 UDER 的梯度解耦或分阶段训练，减少辅助任务竞争。
4. 将 TriCAR 以零残差适配器形式接入 ResNet50，而不是替换已经有效的预训练编码器。
5. 对最终候选模型至少运行多个随机种子，并继续固定 split 3、阈值 0.5 和逐病例 IoU 选模协议。
